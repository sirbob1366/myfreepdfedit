About PDF password protection

This tool encrypts your PDF with AES-256, the same standard used by Adobe Acrobat, using the open-source qpdf engine compiled to WebAssembly and running entirely in your browser. Neither your document nor your password is ever transmitted anywhere. Anyone opening the file will need the password — so store it safely: if you lose it, we cannot recover it, by design.
